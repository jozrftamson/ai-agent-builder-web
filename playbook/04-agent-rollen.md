# Agent-Rollen

## Produktidee

Die Website soll spezialisierte Agenten fuer KI-Entwicklung als klar erkennbare Rollen abbilden.

## Geplante Rollen

### Planning Agent

- zerlegt Ideen in Module, Features und Roadmaps
- erstellt Spezifikationen und Arbeitspakete

### Coding Agent

- erzeugt Boilerplate, Komponenten und API-Strukturen
- hilft bei Refactoring und Implementierung

### Debug Agent

- analysiert Fehler, Logs und reproduzierbare Defekte
- priorisiert Ursachen und Reparaturschritte

### Review Agent

- bewertet Codequalitaet, Risiken und Struktur
- gibt Review-Hinweise und Verbesserungsvorschlaege

### Deployment Agent

- plant Releases, Umgebungen und Rollouts
- verbindet Build, CI und Deployment-Fragen

### Research Agent

- sucht Patterns, APIs, Libraries und Wettbewerbsansatze
- verdichtet Erkenntnisse in Entscheidungen

### Skill Librarian

- pflegt Skill-Entwuerfe, Playbooks und wiederverwendbare Ablaufe
- verbindet Projektwissen mit Agent-Workflows

## Design-Prinzip

Jede Rolle braucht:

- klaren Zweck
- erkennbare Inputs
- klaren Output-Typ
- passende Tools
- Grenzen des Einsatzbereichs
