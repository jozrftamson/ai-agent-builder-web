# Custom Skills

Hier liegen lokale Skill-Entwuerfe fuer dieses Projekt.

Diese Dateien sind absichtlich noch keine finalen Hermes-Skills. Sie sind Arbeitsstaende, die wir erst im Projektalltag testen.

## Regeln

- kurz und konkret schreiben
- echte Inputs und Outputs nennen
- jeden Entwurf spaeter gegen reale Nutzung pruefen
- erst nach mehreren erfolgreichen Anwendungen als echten Skill uebernehmen
