# Agent Modules

## 1. Planning Agent

Uebersetzt Ideen in Aufgaben, Architektur und Roadmaps.

## 2. Coding Agent

Hilft bei Implementierung, Refactoring und Boilerplate.

## 3. Debug Agent

Analysiert Fehler, Logs und reproduzierbare Bugs.

## 4. Review Agent

Prueft Codequalitaet, Struktur und Risiken.

## 5. Deployment Agent

Unterstuetzt bei Build, Release und Rollout.

## 6. Research Agent

Sammelt technische Informationen, Patterns und APIs.
